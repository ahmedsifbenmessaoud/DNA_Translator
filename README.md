# DNA_Translator
High School project
This program converts a given DNA sequence to its RNA and multipeptide, Buit using python and PyQt5.
# install dependencies

```sh
$ pip install pyqt5
$ pip install regex
```

# How to Run

```sh
$ python MainPage.py
```
